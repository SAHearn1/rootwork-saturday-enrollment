# Architecture Map — rootwork-saturday-enrollment

Next.js 14 registration portal. PostgreSQL + Prisma. NextAuth authentication. Stripe payments. Resend transactional email.

## Layers
- UI: Next.js 14 + React 18 + Tailwind CSS
- Auth: NextAuth
- Data: PostgreSQL + Prisma
- Payments: Stripe + @stripe/react-stripe-js
- Email: Resend
- State: Zustand + react-hook-form + Zod
